VintaSoft Web TWAIN JavaScript SDK Files
========================================

To enable scanning functionality, you need to copy the VintaSoft JavaScript files to this folder.

REQUIRED FILES:
- Vintasoft.Shared.js
- Vintasoft.Twain.js

WHERE TO FIND THEM:
1. Download VintaSoft Twain .NET SDK from: https://www.vintasoft.com/download.html
2. After installation, find the JS files in:
   - C:\Program Files\VintaSoft\Twain .NET v15.x\Bin\JavaScript\

   OR from the npm package:
   - node_modules/vintasoft-web-twain-js/dist/

3. Copy Vintasoft.Shared.js and Vintasoft.Twain.js to this folder.

ALSO REQUIRED:
- VintaSoft Web TWAIN Service must be running
- Download from: https://www.vintasoft.com/zip/VintasoftWebTwainService-15.2.2.zip
- Check status at: http://localhost:25319/api/VintasoftTwainApi/Status

LICENSING:
- Evaluation version works for testing
- For production, obtain a license from: https://www.vintasoft.com/vstwain-dotnet-index.html
